## Spring Boot RESTful API Simple Demo

The Demo provide GET, POST, PUT, DELETE, PATCH endpoints

API Endpoints tested via. POSTMAN
- GET 
  - /api/students
  - /api/students/{id}
- POST
  - /api/students
- PUT
  - /api/students/{id}
- DELETE
  - /api/students
  - /api/students/{id}
- PATCH
  - /api/students/{id}
